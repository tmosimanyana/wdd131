Freelance Web Development Portfolio

Site Name

The chosen site name is "Freelance Web Development Portfolio". This name was selected because it clearly represents the purpose of the website, which is to showcase my freelance web development projects and services.

Site Purpose

The purpose of this website is to provide a platform to display my web development skills, past projects, and services I offer. It serves as an online portfolio to attract potential clients by demonstrating my expertise and the quality of my work.

Scenarios

  • How can I view examples of previous web development projects?
  • What web development services does this freelancer offer?
  • How can I contact this web developer for a project inquiry?

Color Schema

The selected colors are:

  • #2c3e50 for headings and footer background
  • #ecf0f1 for background and text sections
  • #3498db for accents, links, and buttons

Typography

The selected fonts are:

  • Open Sans: Used for body text for readability
  • Roboto: Used for headings to create a modern look
