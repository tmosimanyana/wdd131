Site Plan - Botswana Tourism Guide

Site Name

The site name for this project is "Botswana Tourism Guide".

This name was selected because it accurately reflects the purpose of the website, which is to provide comprehensive information and guidance for tourists visiting Botswana.

Site Purpose

The purpose of the "Botswana Tourism Guide" website is to provide tourists with essential information about Botswana, including travel destinations, activities, accommodations, travel tips, and contact details. The website aims to promote tourism in Botswana and assist visitors in planning their trips effectively.

Scenarios

1. A visitor is planning a safari trip to Botswana and wants to know about the best safari destinations, accommodations, and safari tour operators.

2. A family is considering a vacation in Botswana and is looking for information about family-friendly activities, accommodations, and safety tips.

Typography

Font selections:

  • Heading: Arial
  • Body: Roboto

Where each font will be used:

  • Heading: Main headings and navigation links
  • Body: Paragraph text and other content

Color Schema

Primary Color: #336699

Secondary Color: #99CCFF

The primary color will be used for headings and navigation elements, while the secondary color will be used for background elements and accents.
